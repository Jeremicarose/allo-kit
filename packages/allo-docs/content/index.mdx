# Introduction

AlloKit is a modular framework for rapidly building and experimenting with on-chain funding strategies.
It is built on top of **[Scaffold-ETH 2](https://github.com/scaffold-eth/scaffold-eth-2)** and inspired by **[OpenZeppelinâ€™s](https://docs.openzeppelin.com/contracts/5.x/)** design principles of modularity, extensibility, and reuse.

### Key features

- [Components](./allo-app/components) built with **[shadcn/ui](https://ui.shadcn.com)**.
- [Indexer](./indexer) built with **[Ponder](https://ponder.sh)**
- [Strategy Contracts](./contracts) built with **[OpenZeppelin](https://openzeppelin.com)**
- Hooks generated by **[Wagmi CLI](https://wagmi.sh/cli/getting-started)**
